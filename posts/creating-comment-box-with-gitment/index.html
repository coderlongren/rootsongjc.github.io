使用gitment为你的网站添加评论框

Sun Sep 24, 2017

300 Words|Read in about 1 Min
Tags: github-pages  

强烈推荐一个网站评论插件 gitment 。该插件不需要服务端交互，使用 GitHub Issue 存储评论内容，直接在你的博客中引入 JS 代码即可。安装步骤参考 给你的静态博客增加留言系统。

我使用的是 hugo 生成静态网站，同样也是支持的。

以前用过 友言，多说，现在多说已经挂掉了，官方网站 http://duoshuo.com 也无法打开，友言的服务也不稳定，gitment 简直就是我等使用 GitHub Pages 用户的福音。再次🙏 imsun 同学。

想试用 gitment 评论那就赶紧评论一下吧！

PS. 每篇文章的评论功能需要我本人开启，评论归档到了 Issue 下面。
